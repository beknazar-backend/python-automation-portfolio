# Image Processing

Утилита для пакетной подготовки товарных изображений: изменение размера, обрезка, переименование и сохранение в нужном формате.

## Что умеет

- приведение изображений к квадратному формату
- обрезка или вписывание с заполнением
- сохранение в JPEG или PNG
- автоматическое переименование файлов
- поддержка вложенных папок
- режим `dry-run` для проверки результата без записи

## Установка

```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

## Запуск

```bash
python process_images.py -i input -o output
```

Если не указывать аргументы, будут использованы папки `input` и `output` в текущей директории.

## Параметры

- `-i, --input` — папка с исходными изображениями
- `-o, --output` — папка для готовых файлов
- `-s, --size` — целевой размер (по умолчанию `1000`)
- `--mode` — режим обработки: `crop` или `fit`
- `--template` — шаблон имени, например `product_{:03d}`
- `--format` — `jpeg` или `png`
- `--quality` — качество JPEG (по умолчанию `95`)
- `--recursive` — обрабатывать вложенные папки
- `--dry-run` — показать действия без сохранения

## Пример

```bash
python process_images.py -i input -o output --size 1200 --mode crop --format jpeg --quality 90
```

## Советы

- Поместите исходные файлы в папку `input`
- После обработки проверьте `output`
- Если используете PNG, настройка `--quality` не применяется

## Структура проекта

- `process_images.py` — основной скрипт
