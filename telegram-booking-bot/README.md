# Telegram Booking Bot

Telegram-бот для записи клиентов, который собирает заявки, сохраняет их в SQLite и отправляет уведомления администратору.

## Возможности

- прием заявок через Telegram
- валидация имени и номера телефона
- выбор услуги из предложенного списка
- сохранение заявок в `applications.db`
- логирование заявок в `applications.log`
- отправка заявки администратору в реальном времени

## Быстрый старт

### Установка

```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

### Настройка

Скопируйте `.env.example` в `.env` и заполните:

```env
BOT_TOKEN=ваш_токен
ADMIN_CHAT_ID=ваш_admin_chat_id
```

### Запуск

```bash
python main.py
```

## Использование

1. Добавьте бота в Telegram через @BotFather
2. Откройте чат с ботом и отправьте `/start`
3. Укажите имя, телефон и выберите услугу

## Структура проекта

- `main.py` — основной бот
- `config.py` — настройки услуг
- `database.py` — работа с SQLite
- `validators.py` — проверка данных
- `admin_tools.py` — админ-команды и экспорт
- `.env.example` — пример конфигурации
- `requirements.txt` — зависимости

## Важные заметки

- Никогда не публикуйте файл `.env`
- Если бот не отправляет заявку администратору, проверьте `ADMIN_CHAT_ID`
- Убедитесь, что администратор написал боту хотя бы один раз

## Типовые команды

```bash
python main.py
```

## Дополнительно

Для изменения доступных услуг отредактируйте словарь `SERVICES` в `config.py`.
