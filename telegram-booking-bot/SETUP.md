# 📖 Пошаговая инструкция по установке

## Шаг 1: Получение токена бота

1. **Откройте Telegram** на любом устройстве
2. **Найдите** @BotFather (нажмите на поиск 🔍)
3. **Отправьте команду:**
   ```
   /newbot
   ```
4. **Следуйте инструкциям:**
   - Придумайте имя для бота (например: "MyAppBot")
   - Придумайте username (должен заканчиваться на "_bot", например: "my_app_bot")
5. **Скопируйте токен** - выглядит так: `YOUR_BOT_TOKEN_HERE`

## Шаг 2: Получение ID администратора

**Способ 1 (Самый простой):**
1. **Найдите** @userinfobot в Telegram
2. **Он вам сразу покажет ваш ID**

**Способ 2:**
1. Напишите боту какое-то сообщение
2. В логах бота появится `user_id` - это ваш ID

## Шаг 3: Установка Python (если не установлен)

### Windows:
1. Скачайте Python с [python.org](https://www.python.org/downloads/)
2. При установке **обязательно** отметьте ✅ "Add Python to PATH"
3. Завершите установку

### Проверка:
Откройте PowerShell/CMD и выполните:
```bash
python --version
```

Должна появиться версия Python 3.8 или выше.

## Шаг 4: Настройка проекта

1. **Откройте папку проекта** в PowerShell/CMD
   ```bash
   cd D:\Desktop\Works\телеграм бот
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Создайте файл `.env`:**
   
   Способ 1 (Быстро):
   ```bash
   # Скопируйте пример
   copy .env.example .env
   
   # Отредактируйте .env (открыть блокнотом)
   notepad .env
   ```
   
   Способ 2 (Вручную):
   - Откройте текстовый редактор (Блокнот)
   - Напишите:
     ```
     BOT_TOKEN=YOUR_BOT_TOKEN_HERE
     ADMIN_CHAT_ID=123456789
     ```
   - Сохраните как `.env` в папке проекта

## Шаг 5: Запуск бота

1. **В PowerShell/CMD выполните:**
   ```bash
   python main.py
   ```

2. **Вы увидите:**
   ```
   🚀 Бот запущен! Нажмите Ctrl+C для остановки.
   ```

3. **Отлично!** Бот работает! ✅

## Шаг 6: Тестирование

1. **Найдите вашего бота в Telegram** (по username из шага 1)
2. **Нажмите Start или напишите** `/start`
3. **Заполните форму:**
   - Имя: Иван
   - Телефон: +77001234567
   - Услуга: выберите одну из кнопок
4. **Проверьте**, что заявка пришла в ваш чат

## 🔧 Команды администратора

После установки можете использовать команды для просмотра заявок:

```bash
# Просмотреть последние 10 заявок
python admin_tools.py list 10

# Показать статистику
python admin_tools.py stats

# Экспортировать в CSV
python admin_tools.py export

# Справка
python admin_tools.py help
```

## ❌ Если что-то не работает

### Ошибка: "токен в .env не найден"
- Убедитесь, что файл называется `.env` (именно так, с точкой в начале)
- Убедитесь, что он в одной папке с `main.py`

### Ошибка: "python: команда не найдена"
- Python не установлен или не добавлен в PATH
- Переустановите Python, отметив "Add Python to PATH"

### Бот не отправляет заявки администратору
- Проверьте ADMIN_CHAT_ID (должен быть числом)
- Убедитесь, что администратор переписывался с ботом хотя бы раз
- Проверьте корректность токена

### Ошибка при установке зависимостей
```bash
# Попробуйте:
pip install --upgrade pip
pip install -r requirements.txt
```

## 📞 Получение помощи

**Проверьте логи:**
- Логи заявок: `applications.log`
- База данных: `applications.db`

**Примеры для редактирования `config.py`:**

Добавить новую услугу:
```python
SERVICES = {
    'konsultaciya': '💬 Консультация',
    'obsluzh': '🛠️ Обслуживание',
    'новая_услуга': '🎯 Новая услуга',  # ← Добавьте эту строку
}
```

Изменить ID администратора (если нужно перейти на другой аккаунт):
1. Получите новый ID
2. Отредактируйте `.env` файл
3. Перезапустите бот
